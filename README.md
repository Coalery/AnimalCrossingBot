## fishdata.db informations

* Table name : fishdata

* Fields
|name|description|type|
|:---:|:---|:---:|
|name|물고기의 이름|String|
|datestr|물고기가 나타나는 기간|String|
|time|물고기가 나타나는 시간대|String|
|location|물고기가 나타나는 위치|String|
|size|물고기의 크기|String|
|price|물고기의 가격(벨)|Integer|

`민물고기 : 강, 연못, 호수, 절벽위, 하구`
`바닷물고기 : 바다, 부두`

* SQLite
